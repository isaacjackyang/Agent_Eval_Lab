實驗控制台Experiment Control

所有啟動、停止、suite 與 nightly 的操作都集中在這裡，和觀測區塊明確分開。All launch, stop, suite, and nightly controls live here in one dedicated area, clearly separated from the monitoring views.

控制細節Control Detail

批次測試 / SuiteBatch Test / Suite
同一組設定連跑多輪，量測平均分數、通過率與穩定度。Runs the same config across multiple cases to measure mean score, pass rate, and stability.
Nightly 演化Nightly Evolution
逐輪調參找更好的 baseline；若分數提升才採用，並沿用新 baseline 繼續下一輪。Tunes one parameter at a time to find a better baseline; only improved runs are adopted for the next round.
Relayer 掃描Relayer Scan
掃描 relayer layer window 與重複次數，先做 synthetic ranking，再把 top-k 候選送去真實驗證。Scans relayer layer windows and repeat counts, ranks them synthetically first, then verifies the top-k candidates with real evaluation.
